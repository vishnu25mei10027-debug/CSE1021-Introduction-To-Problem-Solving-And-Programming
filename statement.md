Hotel Management System

Problem Statement
Hotels need a simple system to manage room bookings and customer records. Manual work is slow and can cause mistakes. This project helps to manage hotel rooms easily using a simple computer program.

Scope of the Project
This project can be used by:
Small hotels
Hotel staff for room management
Students learning Python

The system works for basic room booking and checkout process.
Target Users
Hotel reception staff
Hotel managers
Students practicing Python programming

High-Level Features
Show available and booked rooms
Book a room for customers
Checkout customers and free the room
View current bookings
